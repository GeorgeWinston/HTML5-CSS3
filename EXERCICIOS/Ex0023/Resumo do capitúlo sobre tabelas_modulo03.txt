++++TABELAS+++++

____Hierarquia de tabelas______

table = tabela
   table row = linha da tabela
	table header = cabeçalho da tabela
	table data = dados da tabela

Obs:
1º _ Seguir sempre a hierarquia das tabelas.O que vem primeiro sempre em primeiro.
2° _ para fazer o alinhamento dos dados de uma tabela usa-se (text-align) para o 
alinhamento horizontal e (vertical-align) para o alinhamento vertical. porém, para
alinhar ao centro com o vertica-align usa-se "middle" ao ínves de "center".
